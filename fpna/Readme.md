# FP&A Calculation Engine

this is the basis for the FP&A project. It contains the basic 

- Data Model, 
- Importers, 
- Utils, 
- Storage, 
- Identity, 
- Calculation Scopes and 
- Report definition 

for the four basic pillars of FP&A toolkit: Earnings, Solvency, Liquidity and Beyond Financials.
